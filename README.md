# AARSHI — The Drama Club Website

Official website for **AARSHI**, the Theatrical and Dramatics Society of IISER Kolkata.

## 🎭 Live Site

Deployed via GitHub Pages at: `https://berasankhadeep20-lang.github.io/AARSHI_WEBSITE/`

## 📁 Structure

```
AARSHI_WEBSITE/
├── index.html               # Single-page website
├── css/
│   └── style.css            # All styles (dark theatrical theme)
├── js/
│   └── main.js              # Curtain animation, nav, scroll reveals
├── .github/
│   └── workflows/
│       └── deploy.yml       # Auto-deploy to GitHub Pages on push to main
└── README.md
```

## 🚀 Deploy

1. Push to the `main` branch.
2. GitHub Actions will automatically deploy to GitHub Pages.
3. Enable GitHub Pages in **Settings → Pages → Source: GitHub Actions**.

## ✏️ Making Changes

- **Content**: Edit `index.html` directly.
- **Colors / fonts**: Change CSS variables at the top of `css/style.css`.
- **Add images**: Drop files into an `assets/` folder and reference them in `index.html`.
